Twitter Micro Services

A project created using the hug framework. Hug exposes API endpoints that allow the user to perform various services (full list of services can be seen in the repo). Some endpoints, such as posting tweets, require user authentication while others are available to everyone. HAProxy ensures that an adequate amount of load is placed into each instance of the available services. The repository below is a prototype that I completed on my own. Code for the complete version is available upon request.

Notflix

Notflix is a movie recommendation site that allows users to browse a catalog of movies for our hypothetical service. The home page receives updates from the Movie DB API thus keeping our selection of popular movies constant with what is currently popular. More details on each movie are available through a click of a button. This project was developed in an agile environment with fellow students at Cal State Fullerton. All the contributors can be seen in the repository's README file. Live site here.

FlapPy Bird

An open-source project that built upon an already established game. The original project used Pygame to recreate Flappy Bird. FlapPy Bird Enhanced took that base and altered the game logic through power ups. Additionally, a new control scheme was added to make the game less infuriating. Instructions to install the game can be found in the repository below.
